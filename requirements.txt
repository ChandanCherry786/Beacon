# Beacon dependencies
# ========================================
#
# PYTHON PACKAGES
# ---------------
# Every package here is OPTIONAL. Each one enables a single feature, and the
# app works without it. The core server uses only the Python standard library.
#
#     pywinpty        Windows pseudo-console (ConPTY) for a real embedded
#                     terminal where interactive tools (claude, vim, tab
#                     completion, colors) work inline. Without it the app
#                     falls back to a simpler line-based terminal.
#     ruff            Formats Python files from Tools > Format Python file.
#                     Black also works if it is installed instead of Ruff.
#     jupyter         Runs .ipynb notebooks (right-click a notebook > Run).
#     nbconvert       The nbconvert engine used to execute those notebooks.
#
# Install the small defaults with:  pip install -r requirements.txt
# Jupyter is larger, so it stays commented below; uncomment it or run
# `pip install jupyter nbconvert` if you work with notebooks.
#
# The terminal front end (xterm.js) is vendored in the vendor/ folder, so it
# needs no download or build.
#
# Python: 3.9+ (CPython, includes tkinter). Verified on 3.13.7.
pywinpty
ruff
# jupyter
# nbconvert
#
#
# EXTERNAL SYSTEM TOOLS (not Python packages; install separately, on PATH).
# Each is optional and only affects its own feature.
#
#   git            git status, commit, pull, push, sync
#   latexmk+tlmgr  LaTeX compilation and automatic package install (TinyTeX
#                  or TeX Live); missing packages install on demand
#   claude         Claude Code CLI, for the Claude assistant panel
#   Microsoft Edge preferred UI window (falls back to the default browser)
#   ollama         optional, free local open-source AI models (ollama.com)
#
# Internet access is needed only for the research finder, package installs,
# and any hosted AI models you choose to use.
